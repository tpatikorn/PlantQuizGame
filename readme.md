## how to setup
1. install requirements.txt
2. create .env e.g.

       DB_SERVER=localhost
       DB_USER=admin
       DB_PASS=SaWasDee_1
       DB_DB=plant_quiz_game
       IMAGE_ROOT=C:\datasets\plant_images\
       DEBUG_MODE=True

3. install npm, if not already
4. install vue through npm using

       npm install vue
